# Terry Roast Plugin

Claude Code plugin packaging for the Terry Roast skill.

## Install from this repository marketplace

1. Add the marketplace:
   `/plugin marketplace add lehzhu/north-york-studio`
2. Install the plugin:
   `/plugin install terry-roast@north-york-studio`

## What it provides

- `skills/terry-roast/SKILL.md` for repo roasts in Claude Code
